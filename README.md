## 🚀 About Me

I’m **Dr. Francis Jesmar P. Montalbo** (**FJP Montalbo**), a **Filipino AI researcher, author, software developer, and Associate Professor** at **Batangas State University, The National Engineering University**. My work sits at the intersection of **artificial intelligence, machine learning, deep learning, computer vision, biomedical signal processing, medical image analysis, data science, scientific computing, and AI-driven software engineering**.

I build **research-grade, reproducible, and deployment-aware AI systems**—from lightweight deep neural networks and transformer-based vision models to intelligent software, data products, and modern AI applications powered by **LLMs, retrieval-augmented generation (RAG), multimodal learning, and scalable backend systems**.

> **Searchable research keywords:** artificial intelligence, AI research scientist, machine learning, deep learning, computer vision, medical AI, biomedical signal processing, medical image analysis, radiology AI, data science, scientific computing, LLMs, large language models, RAG, graph RAG, multimodal AI, vision-language models, knowledge distillation, transformer-based vision models, state space models, software engineering, mobile app development, full-stack development, Batangas State University, Philippines.

## 🌟 Selected Research & Open-Source Highlights

- **[SWAT-DCNN](https://github.com/francismontalbo/swatdcnn)** — automated diagnosis of diverse coffee leaf images using a stage-wise aggregated triple deep convolutional neural network.
- **[MFuRe-CNN](https://github.com/francismontalbo/mfurecnn)** — cost-efficient gastrointestinal disease classification from endoscopy imagery using fused compressed ConvNets and residual learning.
- **[COVID-19 Diagnosis with Truncated DCNNs](https://github.com/francismontalbo/covid19_diagnosis_and_analysis_of_truncated_dcnns)** — lightweight deployable models for chest X-ray and CT-based diagnosis.
- **[Mosquito KD 2021](https://github.com/francismontalbo/mosquito_kd_2021)** — efficient mosquito taxonomy with knowledge distillation, feature fusion, and deployment-oriented model design.
- **[MHADFormer](https://github.com/francismontalbo/mhadformer)** — a cost-efficient hybrid transformer framework for Alzheimer’s disease MRI analysis.
- **[DySARNet](https://github.com/francismontalbo/dysarnet)** — lightweight self-attention deep learning architecture for advanced medical imaging tasks.
- **[S3AR U-Net](https://github.com/francismontalbo/s3ar_unet)** — attention-gated residual U-Net style work for segmentation-oriented biomedical imaging.
- **[Learning Python](https://github.com/francismontalbo/learning_python)** — structured Python learning resources spanning fundamentals to advanced programming and computer science concepts.
